Is it Worth Buying Expensive Wine?

Understanding relationship between perceived score, and price of wine.

By, Shruti Sharma </font>

Introduction



Tasting wine is fun hobby. In fact, it can be converted into a well paying profession as well. A master sommellier can earn up to 150,000 USD a year. However, it can be difficult to achieve as wine can be very expensive. The most expensive wine sold at 304,375 USD was Chteau Cheval Blanc. Compare this to the most expensive champagne sold at 49,000 USD. To those who enjoy wine, is it worth spending more on a fancy bottle of wine rather than simple box wine? What are some chemical factors are important to a wine?

The objective of this project is to get a better understanding about wine price, and quality. This is to see if spending exorbitant money on wine is justified. It also attempts to see if varying traits like acidity, and pH affect the quality of the wine, effect the price of the wine.

Part 1: Corelation between Price and Score

1.1 Reading Data

The data set used to find a corelation between price and taste was created by user zackthoutt [1] by scraping Wine Enthusiast .

It contains reviews for different wines by different sommeliers. It contains the sommellier's description of the wine, along with the score given. It also contains information about the price, variety [Riesling, Pinot Noir, etc.], designation, and the winery it was created in, and the country.

The score given to the wine is scaled out of 100, and is subjective to the reviewer. This provides some form of inconsistency in the data. </font>

# Clean data by removing nan values, and removing duplicates

reviews = reviews.dropna()

# Find duplicates by grouping, and filtering groups based on size
duplicates = reviews.groupby(['designation', 'variety', 'winery']).agg(['count', 'mean']).reset_index()
duplicates.columns = ['designation', 'variety', 'winery', 'points_count', 'points_mean', 'price_count', 'price_mean']
duplicates = duplicates[duplicates['points_count'] > 1]

# Map for mean scores
mean_score = {}
for index, data in duplicates.iterrows():
    mean_score[data['designation'] + data['variety'] + data['winery']] = data['points_mean']

# Substitute with mean scores
reviews['points'] = reviews.apply(lambda data: data['points'] 
                                  if (data['designation'] + data['variety'] + data['winery']) not in mean_score
                                  else mean_score[data['designation'] + data['variety'] + data['winery']], axis=1)

reviews = reviews.drop_duplicates(subset=['designation', 'variety', 'winery'])

# Round points for easier processing
reviews['points'] = reviews['points'].apply(lambda x: round(x))

The pvalue < 0.05 which also proves that there is a corelation.

The graph above shows that change in points as price changes is logarithmic. Thus, the predictive model must be logarithmic.
To make a simplify the calculations, the price was divided into 50 equal buckets and the medial price for each of the buckets was considered. Price buckets with no values were dropped.

Since the price was standardized, there were few values which were negative. To account for that, the absolute value was taken. </font>

# Plot trend between price and points

price_groups = reviews.groupby(pd.cut(reviews['price'], bins=50)).agg(np.median)
price_groups = price_groups.dropna()

X = price_groups['price'].values
y = price_groups['points'].values 

# Account for negative standardized price
X = np.apply_along_axis(lambda x: abs(x), axis=0, arr=X)

trend = np.polyfit(np.log(X), y, 1)
y_predicted = trend[0]*np.log(X) + trend[1]

fig, ax = plt.subplots()
ax.plot(X, y, 'o', X, y_predicted)
ax.grid()
ax.set_xlabel('Price')
ax.set_ylabel('Points')
ax.set_title('Price vs. Points')

As seen by the scattered data points (in blue), and the trend line (in orange), there is a corelation between price and score received by a proffessional sommellier. However, the graph tends to taper at higher prices before reaching the perfect score of 100.

Therefore, it can be concluded that higher priced wine do tend to be better, but one does not need to break the bank to taste the best. </font>

# Create a word cloud

all_descriptions = pd.read_csv('winemag-data-130k-v2.csv', usecols=['description', 'points'])
all_descriptions = all_descriptions.nlargest(50, 'points')

comment_words = ''
stopwords = set(STOPWORDS)
more_stopwords = ['will','new','now','year', 'least', 'vineyard', 'years', 'drink', 'yet']
for word in more_stopwords:
    stopwords.add(word)

for _, row in all_descriptions.iterrows(): 
    tokens = row['description'].split() 
       
    for words in tokens: 
        comment_words = comment_words + words.lower() + ' '


wordcloud = WordCloud(width = 800, height = 600, max_words=50,
                background_color ='white', 
                stopwords = stopwords, 
                min_font_size = 10).generate(comment_words)
plt.figure(figsize = (8, 6))
